# Einstein Puzzle

A logic game inspired by the [Einstein puzzle](https://en.wikipedia.org/wiki/Zebra_Puzzle), implemented using web technologies.

This version of the game is heavily inspired by this [C++ version](https://web.archive.org/web/20121029043853/http://games.flowix.com/en/index.html), keeping the same logic, rules, and game play, but reimplemented in a web-based environment using TypeScript, HTML, and CSS.
